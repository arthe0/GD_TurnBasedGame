Идея: многопользоватлеьская игра в режиме все против всех на 2-4 человека.
Карта представляет собой гексогональную сетку.

Каждый ход имеет три фазы на каждой из которых персонаж может совершить одно действие или пропустить фазу. Таким образом, на каждом 
ходу игрок выбирает последовательность из 3 действий.

Персонажи имеют шкалу энергии, на которую каждый ход могут применить одну из 3 типов способностей:
Перемещение  (работет по аналогии с цивилизацией 5)

Атака, 3 вида: 
	1. Точный выстрел в противника на средней дистанции (всегда попадает в противника, если он не за препятствием)
	2. Мощный выстрел, выбирается любая клетка для атаки, снаряд падает на клетку через один ход
	3. Ближний бой, удар размахом по 3 ближайшим клеткам в заданном направлении

Абилка, 3 вида:
	1. Возможность поставить стенку на 2 хода (при выборе точкой перемещения этой клетки другим игроком, 
	он останавливается перед ней)
	2. Поставить мину в радиусе 1 клетки (врагам видна только анимация каста спосбности), мина наносит урон
	3. Войти в невидимость на 3 фазы (враг видит вас только в радиусе 2-х клеток), атака прерывает невидимость
